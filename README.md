# 停车场管理系统简介 java1033

## 项目概述

本项目是一款基于B/S架构的停车场管理系统，采用MVC模式进行设计。系统实现了前后端分离，前端使用Vue框架，后端基于SpringBoot构建，结合Mybatis-Plus进行数据库操作。

## 技术栈

- **开发语言**：Java
- **数据库**：Mysql
- **后端框架**：SpringBoot、Mybatis-Plus
- **前端框架**：Vue
- **开发工具**：IDEA、JDK1.8、Maven

## 功能模块

### 系统角色

- **普通用户**：个人信息管理、预定停车位、缴费信息管理
- **管理员**：用户信息管理、车辆信息管理、停泊车辆管理、车辆进出管理
- **教师**：考试管理、题目管理、查询学生成绩、学生管理
- **学生**：考试中心、考试答题、提交试卷、查看成绩、交流区

### 功能描述

- **普通用户**：拥有基本操作权限，可管理个人信息和停车位缴费等。
- **管理员**：负责系统用户和车辆信息的管理，以及对停泊车辆和车辆进出的管理。
- **教师**：可管理考试相关内容，包括题目的增删改查和自动组卷功能，以及学生成绩的查询。
- **学生**：参与考试、查询成绩以及在交流区互动。

## 运行环境

- **操作系统**：Windows
